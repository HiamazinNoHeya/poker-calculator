# poker-calculator

ポーカーの計算ツールです。ポットオッズ、α（ブラフ頻度しきい値）、MDF（最小ディフェンス頻度）などを即時計算できます。

## 機能

- ポットオッズ計算
- α（ブラフ頻度しきい値）計算
- MDF（最小ディフェンス頻度）計算
- value/bluff 比の表示
- √α, 1-√α の計算

## 使い方

1. [index.html](index.html) をブラウザで開きます。
2. 「ポットサイズ」と「ベット額」を入力します。
3. 結果が自動で表示されます。
4. 「初期化」ボタンで入力をリセットできます。
