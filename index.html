結果

ポットオッズ

—

式：bet / (pot + 2*bet)

value / bluff

—

式：value% : bluff%

α

—

式：bet / (pot + bet)

MDF

—

式：pot / (pot + bet) = 1 - α

√α

—

式：√α

√αのMDF

—

式：1 - √α

メモ

  • ポットオッズは「コールが利益0になるために必要な勝率」。
  • αは相手が最適にブラフできる上限割合の指標。MDFはこちらが降りすぎないための最低防衛率。
  • 単位はすべて同じ通貨でOK（$や¥が混在しなければ良い）。
